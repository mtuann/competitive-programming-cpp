# Topic Template

Use this shape for mature topic pages in `topics/`.

## 1. Summary

- what the technique solves
- when to suspect it
- typical constraints or patterns

## 2. Core Idea

- one short intuitive explanation
- one proof sketch or correctness invariant

## 3. Complexity

- time complexity
- memory complexity
- common tradeoffs

## 4. Canonical C++ Pattern

- data structure or state layout
- clean implementation outline
- common implementation pitfalls

## 5. Recognition Cues

- what clues in a problem statement point to this topic
- what nearby topics it is often confused with

## 6. Practice Ladder

- `warm-up`
- `core`
- `stretch`

## 7. Go Deeper

- `Course`: official or course-primary link
- `Reference`: CP-Algorithms / OI Wiki / project-primary link
- `Practice`: CSES / AtCoder / USACO Guide / ICPC archive / official platform
- `Paper / notes / blog`: optional classic paper, lecture notes, or strong community write-up

## 8. Related Topics

- prerequisites
- next topics
- reusable reductions or connections
