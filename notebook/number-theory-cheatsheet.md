# Number Theory Cheatsheet

## Core Tools

- `gcd(a, b)`
- `lcm(a, b) = a / gcd(a, b) * b`
- prime factorization
- SPF sieve
- fast power
- modular inverse

## Prime-Exponent View

```text
gcd: take the smaller exponent
lcm: take the larger exponent
```

## Typical Signals

- divisibility
- repeated factorizations
- counting modulo a prime
