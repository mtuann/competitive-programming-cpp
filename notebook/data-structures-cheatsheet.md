# Data Structures Cheatsheet

## Choose By Operations

- static range sums: prefix sums
- point add + prefix/range sum: Fenwick
- point/range updates with associative merge: segment tree
- static idempotent range query: sparse table
- connectivity merges: DSU
- top / min / max only: heap
- predecessor / successor / erase: set or multiset

## First Questions

- static or dynamic?
- prefix-only or arbitrary range?
- online or offline?
- duplicates matter?
