# Combinatorics Cheatsheet

## Common Moves

- choose positions
- split into runs
- distribute identical items into slots
- count bad cases and subtract

## Standard Forms

```text
x1 + x2 + ... + xn = S
xi >= 0
xi >= 1
xi <= K
```

## Questions To Ask

- are objects distinguishable?
- are slots distinguishable?
- is order important?
- is the target exact, at least, or at most?
