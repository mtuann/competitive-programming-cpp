# Contest Playbooks

Contest playbooks are for `how to train`, not just what to solve.

Each playbook should be tied to a specific contest style such as `USACO`, `IOI/OI`, or `ICPC`, and should answer:

- what skills matter most
- how to structure practice
- what to review before a contest
- how to upsolve afterward

Current state: this folder is still a scaffold. It should stay small and practical, with one real guide per contest track before adding more.
