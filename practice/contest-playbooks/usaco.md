# USACO Playbook

## What Matters Most

- clean implementation
- strong data structures and graph basics
- deliberate progression through bronze -> silver -> gold style patterns

## Training Rhythm

- one topic ladder at a time
- short review after every solved set
- keep template use light until logic is stable

## Before A Contest

- re-read the relevant topic pages
- skim notebook formulas and edge cases
- warm up with one implementation problem

## After A Contest

- upsolve every missed problem
- note the first modeling mistake, not just the final algorithm
