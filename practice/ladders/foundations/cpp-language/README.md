# C++ Language Ladder

This ladder is about implementation comfort, not hard algorithms.

## Warm-Up

- simple input / output tasks
- array and vector manipulation
- pair / struct sorting

## Core

- frequency counting with `map` or `unordered_map`
- stack / queue / deque exercises
- comparator-based sorting

## Stretch

- write the same solution twice: once quickly, once cleanly
- compare `vector`, `set`, `map`, and `priority_queue` choices on the same problem
