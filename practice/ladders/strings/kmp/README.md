# KMP Ladder

## Who This Is For

Use this ladder when you want exact linear matching and a stronger grip on borders, prefixes, and fallback logic.

## Warm-Up

- prefix-function construction
- single-pattern matching

## Core

- borders
- repeated prefixes
- fallback logic

## Exit Criteria

You are ready to move on when you can:

- build `pi` from memory
- explain what `pi[i]` means without hand-waving
- scan a text with KMP without moving the text pointer backward
- use border chaining for a non-matching problem
